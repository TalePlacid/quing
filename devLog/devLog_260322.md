### 작업 2. NavigationState 추가

#### 1. 문제
- GUI를 처리하려니 현재 위치한 Task 대한 데이터 필요.
#### 2. 고민
- 현재 위치를 나타내려면 Task`*`가 가장 직관적임.
- 하지만 포인터를 바로 주는 것은 안정성이 떨어짐.
- 관계를 소유하고 잇는 것은 Task가 아니라 Node이고, Node는 ChildArrayForest의 내부 구현이기 때문에 Task를 직접 소유하는 데 특별한 이점도 없음.
#### 3. 해결방안
- NavigationState를 추가.
- Task의 식별자인 uuid를 소유.
#### 4. 작업 내용
- NavigationState 추가.
#### 5. 결과
- GUI 추가 필요.
